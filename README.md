JUEGO DE LA SERPIENTE INICIAL
INTORDUCCIÓN
El presente proyecto se basa en entendimiento de la introducción a la lógica de programación y sus funcionalidades las cuales nos sirven de guía para crear una aplicación simple pero detallada con lo aprendido en clases, el presente documento se basará en las funcionalidades del programa (código), objetivo del programa y la fecha, así mismo, el documento tendrá una recopilación de las librerías utilizadas en el desarrollo del programa.
Funcionalidades:
Para el presente informe, se debía tomar una aplicación para desarrollar, en mi caso elegí el juego de la serpiente la cual es relativamente sencilla en los comandos y guías que pude encontrar y con el aprendizaje de las clases semanales, por ejemplo:

Este ejemplo muestra la configuración de la parte inicial del juego el cual es la cabeza de la serpiente que usa los comandos de agregar (appened) pero en lugar de este término, se tiene el nombre a modificar y el qué se va a modificar.
Otra funcionalidad es las listas las cuales, se usan para agregar el movimiento de la serpiente dentro del juego mediante coordenadas y el lugar al que se espera llegar, así mismo, la configuración de lista elimina los elementos de la serpiente (cuerpo) para crear una nueva estructura:

Adicional, en el ejemplo anterior del movimiento del cuerpo, se utilizan las variables “For” e “If” que nos ayudan a regular el cuerpo con un segmento relacionado entre los detallados en el sistema y el “If” es cuando es superior a cero, el valor del cuerpo cera cero (inicial) en el tema del movimiento del cuerpo con el movimiento que se requiere.
Finalmente, se utiliza la variable “While” el cuál, se utiliza para generar un bucle o ciclo para que reaparezca la comida de la serpiente, que genere el movimiento, que la serpiente pueda morder la comida y que se pueda reiniciar el juego como se puede ver a continuación:
